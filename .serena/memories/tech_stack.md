# Technology Stack

## Core Framework
- **Next.js 15** (15.4.0-canary.38) - React framework with App Router and Turbopack
- **React 19** - UI library with concurrent features
- **TypeScript 5** - Type safety and development experience

## UI and Styling
- **Material-UI v7** (@mui/material, @mui/icons-material, @mui/lab)
- **Emotion** - CSS-in-JS styling (@emotion/react, @emotion/styled, @emotion/cache)
- **Tailwind CSS v4** - Utility-first CSS framework
- **Styled Components** - Additional styling option
- **Material Icons** - Icon system (@fontsource/material-icons)
- **Roboto Font** - Typography (@fontsource/roboto)

## State Management & Data Fetching
- **Zustand** - Lightweight state management
- **React Query (@tanstack/react-query)** - Server state and caching
- **NextAuth.js v5 beta** - Authentication and session management
- **React Query Persist Client** - State persistence

## Data Validation & Processing
- **Zod** - Schema validation and type inference
- **neverthrow** - Functional error handling
- **fast-xml-parser** - XML/RSS feed parsing
- **cheerio** - HTML parsing and manipulation
- **camelcase-keys** - Object key transformation

## Development Tools
- **Vitest** - Unit testing framework with browser testing
- **Playwright** - E2E testing and browser automation
- **Storybook** - Component documentation and testing
- **ESLint** - Code linting with Next.js config
- **Prettier** - Code formatting
- **Monaco Editor** - Code editing in browser

## APIs and External Services
- **OpenMeteo SDK** - Weather data (@openmeteo/sdk)
- **YouTube RSS API** - Video feed integration
- **Reddit JSON API** - Post aggregation
- **RSS/Atom feeds** - Generic feed parsing

## Build and Development
- **Turbopack** - Fast bundler for development
- **Bun** - Used for configuration watching
- **Node.js 20** - Runtime environment

## Utility Libraries
- **ts-pattern** - Pattern matching for TypeScript
- **use-debounce** - React hooks for debouncing
- **lru-cache** - Caching implementation
- **tslog** - Structured logging
- **yaml** - YAML parsing
- **cross-fetch** - Fetch polyfill
- **chokidar** - File watching
